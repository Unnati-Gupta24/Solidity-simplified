## âš™ï¸ 1. What is `pragma` in Solidity?

`pragma` is used to specify **compiler instructions**.  
It tells the compiler which Solidity version should be used for this code.

### âœ… Example:
```solidity
pragma solidity ^0.8.0;  // Only compiles with version 0.8.0 or higher but below 0.9.0

contract VersionExample {
    string public message = "Hello Solidity!";
}
````

### ðŸ’¡ Notes:

* `^0.8.0` means any version >= 0.8.0 and < 0.9.0
* You can also lock to a specific version:

  ```solidity
  pragma solidity 0.8.17;
  ```
* Always specify `pragma` at the **top of your file**.

## ðŸ§© 2. Data Types in Solidity

Solidity is **statically typed**, meaning you must declare the type of each variable.

### ðŸ”¹ Value Types

Stored directly in memory - independent copies.

| Type      | Description                           | Example                       |
| --------- | ------------------------------------- | ----------------------------- |
| `uint`    | Unsigned integer (non-negative)       | `uint age = 25;`              |
| `int`     | Signed integer (positive or negative) | `int temp = -5;`              |
| `bool`    | Boolean value (`true` / `false`)      | `bool isActive = true;`       |
| `address` | Ethereum wallet address               | `address owner = msg.sender;` |
| `bytes`   | Fixed-length binary data              | `bytes32 data = "0x123...";`  |

---

### ðŸ”¹ Reference Types

These store **references (pointers)** to data.

| Type      | Description      | Example                                    |
| --------- | ---------------- | ------------------------------------------ |
| `string`  | Text data        | `string name = "Alice";`                   |
| `array`   | Ordered list     | `uint[] public numbers;`                   |
| `struct`  | Custom data type | `struct Person { string name; uint age; }` |
| `mapping` | Key-value pairs  | `mapping(address => uint) balances;`       |

---

### âœ… Example: Using Different Data Types

```solidity
pragma solidity ^0.8.0;

contract DataTypesExample {
    uint public age = 30;
    bool public isActive = true;
    string public name = "Alice";
    address public owner = msg.sender;
    bytes32 public hashData = keccak256(abi.encodePacked("Hello"));

    uint[] public numbers = [1, 2, 3];
    mapping(address => uint) public balances;

    struct Student {
        string name;
        uint grade;
    }

    Student public student = Student("Bob", 90);
}
```
